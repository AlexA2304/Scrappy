These are the existing endpoints for the Scrappy App

Vote Service

Supports voting actions and data retrieval

  • /vote/findAll: Returns the entire vote table

  • /vote/findByID/{storyID}: Returns the votes by storyID

  • /vote/doVotesMatch/{storyID}/{numberOfVoters}: Returns true if all votes in the storyID are equal AND the number of votes equals the provided number of voters

  • /vote/findByUsername/{username}: Returns the vote table for a single username

  • /vote/findByVote{vote value}: Returns all votes with the same value provided

  • /vote/insert: Insert a new value into the vote table

    fetch POST example:

    {

    storyID: {number}

    username: {string}

    vote: {1, 2, 3, 5, 8, 13}

    }

  • /vote/update: Update an existing vote

    fetch POST example:

    {

    storyID: {number}

    username: {string}

    vote: {1, 2, 3, 5, 8, 13}

    }

    • storyID
    • username
    • vote
